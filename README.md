# JotParty

Demo completa de "Tinder de trabajos para eventos sociales" construida con Next.js 14 (App Router), Tailwind CSS y Framer Motion. Todo funciona con datos locales y `localStorage`, sin backend.

## Rutas clave
- `/` Home con feed de eventos y modal rápido.
- `/swipe` Experiencia tipo Tinder con swipe, match y confetti.
- `/messages` Bandeja y chat con respuestas automáticas.
- `/profile` Perfil editable, skills y reset de demo.
- `/assistant` Guía en chat con chips de preguntas rápidas.
- `/login` Inicio de sesión demo (solo localStorage).

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

> Nota: el entorno de este repositorio usa solo frontend y mocks. No necesitas backend para probarlo.
